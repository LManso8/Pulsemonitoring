# Project Decisions

## Sensor Selection

### Initial Solution

Geophone

### Final Solution

Accelerometer

### Reason

Lower cost, easier integration and improved portability.

---

## Data Format

### Initial Format

JSON

### Final Format

CSV

### Reason

Simpler storage, processing and interoperability.

---

## Operating Modes

Two operating modes were implemented:

### Environmental Mode

Assessment according to technical standards.

### Seismic Mode

Detection and classification of seismic events.

---

## Enclosure

A 3D printed enclosure was added to improve system protection, portability and usability.
